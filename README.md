# Clase NÂ° 37 - Dividir en capas nuestro proyecto
Para esta entrega utilice como arquitectura el modelo vista controlador como base modificado, utilizando domain driven design, dividiendo asi el proyecto en capas de ruteo, controlador, logica de negocio y persistencia de datos.
